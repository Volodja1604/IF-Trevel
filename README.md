# IFtravel

## Сайт IFtravel - туристичний Івано-Франківськ ![IFTravel](http://webui-teamx.github.io/IFtravel/team/image/logo.png)

[Перейти на сайт IFtravel](http://webui-teamx.github.io/IFtravel/ "Перейти на сайт IFtravel")

### Ідея та призначення сайту

Ідея сайту - турист, який бажає відвідати Івано-Франківськ, але нічого про місто не знає, зайшовши на наш сайт отримує головну інформацію:

* Коротку довідку про місто 
* Що варто побачити в місті в першу чергу
* Де зупинитись на проживання
* Де смачно поїсти
* Новини  про актуальні цікаві події в місті
